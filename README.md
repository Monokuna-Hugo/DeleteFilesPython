# 文件清理工具套件

基于wxPython开发的图形用户界面(GUI)应用程序套件，包含两个工具：

1. **基础文件删除工具** - 按文件后缀批量删除文件
2. **高级文件清理工具** - 无后缀文件识别和清理，包含安全删除功能

## 工具对比

| 功能特性 | 基础工具 | 高级工具 |
|---------|---------|---------|
| 按后缀删除 | ✅ | ✅ |
| 无后缀文件清理 | ❌ | ✅ |
| 回收站安全删除 | ❌ | ✅ |
| 白名单保护 | ❌ | ✅ |
| 递归扫描 | ❌ | ✅ |
| 隐藏文件处理 | ❌ | ✅ |
| 选项卡界面 | ❌ | ✅ |

## 功能特点

### 基础文件删除工具
- 🖱️ 直观的图形用户界面
- 📁 文件夹路径选择
- 🔍 多文件后缀支持（逗号分隔）
- 👁️ 文件预览和确认机制
- ⚠️ 安全删除确认对话框
- 📊 详细的文件统计信息
- 📝 完整的操作日志记录
- 🛡️ 错误处理和权限检查

### 高级文件清理工具（新增功能）
- 🔍 **无后缀文件识别**：智能识别没有文件扩展名的文件
- 🗑️ **回收站安全删除**：可选择将文件移动到回收站而非永久删除
- 🛡️ **白名单保护**：自动排除重要系统目录，防止误删
- 📂 **递归扫描**：支持深度扫描子目录
- 👻 **隐藏文件处理**：可选是否包含隐藏文件
- 📋 **选项卡界面**：按后缀删除和无后缀清理分开管理
- 📊 **详细统计**：显示文件数量、大小和路径信息

## 安装要求

### 系统要求
- Windows 7/8/10/11
- Python 3.6+

### 依赖安装

```bash
pip install -r requirements.txt
```

或者直接安装wxPython：

```bash
pip install wxPython
```

## 使用方法

### 基础文件删除工具

```bash
python file_deleter_app.py
```
或双击运行 `run.bat`

**操作步骤**：
1. **选择文件夹**：点击"浏览..."按钮选择目标文件夹
2. **输入文件后缀**：在文件后缀输入框中输入要删除的文件类型（如：`.txt,.log,.tmp`）
3. **扫描文件**：点击"扫描文件"按钮查看匹配的文件
4. **预览文件**：在文件列表中查看将要删除的文件详情
5. **确认删除**：点击"执行删除"按钮，在确认对话框中确认操作
6. **查看结果**：在日志区域查看删除操作的详细结果

### 高级文件清理工具

```bash
python advanced_file_cleaner.py
```
或双击运行 `run_advanced.bat`

**无后缀文件清理操作步骤**：
1. **选择扫描目录**：在"无后缀文件清理"选项卡中选择目标目录
2. **配置白名单**：查看或添加需要排除的目录到白名单
3. **设置扫描选项**：选择是否递归扫描和包含隐藏文件
4. **扫描无后缀文件**：点击"扫描无后缀文件"按钮
5. **预览文件列表**：查看识别到的无后缀文件详情
6. **选择删除方式**：勾选"移动到回收站"进行安全删除
7. **执行清理**：点击"清理文件"按钮确认操作
8. **查看详细日志**：在底部日志区域查看操作记录

### 界面说明

- **文件夹路径**：显示当前选择的文件夹路径
- **文件后缀**：支持多个后缀，用逗号分隔
- **文件列表**：显示匹配的文件名、大小和修改时间
- **统计信息**：显示找到的文件数量和总大小
- **操作日志**：实时显示所有操作的详细日志

## 安全特性

- 🔒 **确认对话框**：删除前显示文件列表和数量，防止误操作
- 🚫 **权限检查**：自动检测文件访问权限
- 📋 **详细日志**：记录所有操作，便于审计
- ⚡ **错误处理**：友好的错误提示和恢复机制

## 注意事项

⚠️ **重要提醒**：

1. 删除操作不可逆，请谨慎使用
2. 建议先扫描预览文件，确认无误后再执行删除
3. 程序会检查文件权限，无权限的文件将跳过删除
4. 所有操作都会记录在日志文件中
5. 建议在测试环境中先试用

## 技术支持

如果遇到问题，请检查：

1. Python版本是否符合要求
2. wxPython是否正确安装
3. 是否有足够的文件操作权限
4. 查看日志文件 `file_deleter.log` 获取详细信息
