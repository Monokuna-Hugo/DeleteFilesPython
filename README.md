# 文件删除工具

基于wxPython开发的图形用户界面(GUI)应用程序，用于安全地批量删除指定后缀的文件。

## 功能特点

- 🖱️ 直观的图形用户界面
- 📁 文件夹路径选择
- 🔍 多文件后缀支持（逗号分隔）
- 👁️ 文件预览和确认机制
- ⚠️ 安全删除确认对话框
- 📊 详细的文件统计信息
- 📝 完整的操作日志记录
- 🛡️ 错误处理和权限检查

## 安装要求

### 系统要求
- Windows 7/8/10/11
- Python 3.6+

### 依赖安装

```bash
pip install -r requirements.txt
```

或者直接安装wxPython：

```bash
pip install wxPython
```

## 使用方法

### 运行应用程序

```bash
python file_deleter_app.py
```

### 操作步骤

1. **选择文件夹**：点击"浏览..."按钮选择目标文件夹
2. **输入文件后缀**：在文件后缀输入框中输入要删除的文件类型（如：`.txt,.log,.tmp`）
3. **扫描文件**：点击"扫描文件"按钮查看匹配的文件
4. **预览文件**：在文件列表中查看将要删除的文件详情
5. **确认删除**：点击"执行删除"按钮，在确认对话框中确认操作
6. **查看结果**：在日志区域查看删除操作的详细结果

### 界面说明

- **文件夹路径**：显示当前选择的文件夹路径
- **文件后缀**：支持多个后缀，用逗号分隔
- **文件列表**：显示匹配的文件名、大小和修改时间
- **统计信息**：显示找到的文件数量和总大小
- **操作日志**：实时显示所有操作的详细日志

## 安全特性

- 🔒 **确认对话框**：删除前显示文件列表和数量，防止误操作
- 🚫 **权限检查**：自动检测文件访问权限
- 📋 **详细日志**：记录所有操作，便于审计
- ⚡ **错误处理**：友好的错误提示和恢复机制

## 注意事项

⚠️ **重要提醒**：

1. 删除操作不可逆，请谨慎使用
2. 建议先扫描预览文件，确认无误后再执行删除
3. 程序会检查文件权限，无权限的文件将跳过删除
4. 所有操作都会记录在日志文件中
5. 建议在测试环境中先试用

## 技术支持

如果遇到问题，请检查：

1. Python版本是否符合要求
2. wxPython是否正确安装
3. 是否有足够的文件操作权限
4. 查看日志文件 `file_deleter.log` 获取详细信息
